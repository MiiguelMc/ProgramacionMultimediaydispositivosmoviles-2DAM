# 📱 Módulo de Aplicaciones Multiplataforma y Móviles  
### Desarrollo en Kotlin – Repositorio de Ejercicios y Proyectos

Bienvenido/a a este repositorio, donde se recopilan los **recursos, ejercicios, prácticas y proyectos** desarrollados en el módulo de **Aplicaciones Multiplataforma y Móviles**, con un enfoque principal en el lenguaje **Kotlin**.

Este repositorio sirve como base de aprendizaje y referencia para comprender los fundamentos del desarrollo móvil, especialmente en el ecosistema Android, así como conceptos avanzados relacionados con interfaces, gestión de datos, arquitectura y buenas prácticas de programación.

---

## 🚀 Objetivos del Módulo

- Comprender el uso de **Kotlin** como lenguaje principal para el desarrollo móvil.
- Conocer la estructura y funcionamiento de aplicaciones Android.
- Trabajar con **layouts**, navegación, componentes gráficos y manejo de eventos.
- Aprender a gestionar datos con **SharedPreferences, archivos, bases de datos (Room)** y APIs.
- Introducir patrones de arquitectura como **MVVM**.
- Desarrollar aplicaciones multiplataforma usando librerías o herramientas compatibles (opcional).
- Crear proyectos funcionales y escalables.

---

## 📂 Contenido del Repositorio

Este repositorio suele incluir:

- **/ejercicios**  
  Ejercicios básicos de sintaxis Kotlin, control de flujo, POO, colecciones, etc.

- **/prácticas**  
  Pequeñas aplicaciones Android con distintos niveles de complejidad.

- **/proyectos**  
  Aplicaciones completas aplicando arquitectura, persistencia y diseño moderno.
